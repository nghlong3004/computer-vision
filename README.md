## Cấu trúc Repository

Dự án được tổ chức theo cấu trúc **Monorepo**, trong đó mỗi bài tập thực hành là một thư mục độc lập với môi trường và mã nguồn riêng biệt.

```text
COMPUTER-VISION/
├── basic_image_processing/     # Bài tập 1: Xử lý ảnh
├── abcxyzt/                    # Bài tập 2: abcxyz
.
.
.
├── final_project/              # Đồ án cuối kỳ
└── README.md                   # Tài liệu tổng quan (File này)
```
## Cấu trúc chuẩn của từng bài tập
Mỗi thư mục bài tập con:

```text
project_name/
├── assets/                 # Chứa tài nguyên (Ảnh input, video, ...)
├── src/                    # Mã nguồn chính 
├── main.py                 # File chạy chương trình
├── requirements.txt        # Danh sách thư viện cần thiết cho bài tập này
└── README.md               # Hướng dẫn chi tiết riêng cho bài tập
```
